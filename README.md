# Claude Rule

Standalone rule project for Claude and Claude Code traffic.

## Files

- `dist/Claude.list`: default recommended Surge ruleset
- `dist/Claude_Strict.list`: safer Surge ruleset with lower false-positive risk
- `dist/Claude_Full.list`: broader Surge ruleset with extra third-party telemetry and IP fallback
- `dist/clash/Claude.yaml`: default recommended Clash ruleset
- `dist/clash/Claude_Strict.yaml`: safer Clash ruleset with lower false-positive risk
- `dist/clash/Claude_Full.yaml`: broader Clash ruleset with extra third-party telemetry and IP fallback

## Usage

Use one of the published raw file URLs from the `dist/` directory in your client.
